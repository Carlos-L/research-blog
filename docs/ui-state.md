# UI State

Current visual direction:

- Pure white surface with blue accents.
- No yellow text highlights.
- Simple top navigation with `Home`, `Daily`, `Notes`, `Search`, `About`.
- Home page uses a large research-style hero plus recent content cards.
- Daily and Notes use custom landing pages instead of Quartz default folder listings.
- Markdown content keeps Obsidian-compatible features, but the presentation is kept minimal and academic.

This note captures the current baseline before further visual refinement.
